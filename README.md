# ✈️ Flight Operations Analytics (SQL Project)

## 📌 Project Overview
This project analyzes airline and airport operations data to uncover passenger traffic patterns, route performance, flight frequency, and seat utilization. The goal is to derive actionable insights for improving route efficiency and capacity planning.

---

## 🎯 Objectives
- Analyze passenger traffic across different routes  
- Identify busiest and least busy routes  
- Evaluate average passengers per flight  
- Detect high-traffic corridors between cities  
- Analyze airline seat utilization efficiency  
- Identify most popular destination airports  
- Examine relationship between city traffic and airport activity  

---

## 🛠️ Tools & Technologies
- SQL  
- MySQL  
- Common Table Expressions (CTEs)  
- Joins & Aggregations  
- GROUP BY  
- Data Analysis Techniques  

---

## 📊 Key Analyses Performed
- Passenger Traffic Analysis  
- Route Performance Analysis  
- Flight Frequency Analysis  
- Airport Performance Analysis  
- Seat Utilization Analysis  
- Destination Popularity Analysis  
- City vs Passenger Traffic Analysis  

---

## 💡 Business Insights
- Identified busiest flight routes with highest passenger traffic  
- Discovered high-traffic corridors between major cities  
- Highlighted top-performing airports based on passenger volume  
- Evaluated airline seat utilization efficiency  
- Identified most popular destination airports  
- Provided insights for route optimization and capacity planning
